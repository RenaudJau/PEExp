Mini modle de comptition et tests de coexistence

Ceci est un test de modle pour dessiner les hypothses gnrales de EP. Lide est de faire un petit modle de comptition trs simplifi pour voir la forme que devrait avoir les grandes zones du schma (coexistance ou non..). On modlise la biomasse des 2 espces en faisant plusieurs postulats trs grossiers (mais cest vraiment juste pour voir comment a ragit) :

  • Dans les meilleures conditions (pas de comptition et optimum abiotique) : biomasse = 100
  • Si la distance loptimum est de 10, on retire 10%, donc biomasse = 90
  • Si la comptition exerce par lautre espce est de 10, on retire encore 10 ce qui reste, donc ici biomasse = 80
  • On considre quil y a coexistence si les 2 espces ont une biomasse > 0..
  • Si le semis est squenc, lespce qui arrive en premire subit 2 fois moins de comptition (elle est plus forte..) et elle exerce 2 fois plus de comptition (elle est plus forte jai dit..)

Le raisonnement est trs binaire, mais il me semble que pour dessiner les tendances, cest assez utile..

NB : Du coup, jai du diviser par deux les effets comptitifs, sinon ils sont strictement symtriques de la distance loptimum et il ny a plus de coexistence..

#Distance aux optimum de A et B (ils voluent en sens inverse)
#il y a une rptition pour avoir des points dans les 2 dimensions (quidistance et assymtrie)
DB <- rep(seq(from = 0, to = 100, by = 1),101)
DA <- rep(seq(from = 100, to = 0, by = -1),101) 

#Comptition exerce par A sur B et inversement
CA <- rep(seq(from = 0, to = 50, by = 0.5), each=101)
CB <- rep(seq(from = 50, to = 0, by = -0.5), each=101)

Coexistence_plot2 <- function(DA,DB,CA,CB, xlim=c(-1,2), main = "Effets de priorit", coul = c("#00C3FC","#5AD6FA","#ADDEED","#CFCFCF","#D7CA97","#FFE88C","#FAC901"),...)
{
  #Equidistance
  Equi <- (DB-DA)/(DB+DA)
  
  #Assymtrie :
  Assy <- (CA-CB)/(CB+CA)
  
  #Calcul des biomasses des espces
  spA <- 100-DA-(CB)
  spB <- 100-DB-(CA)
  
  #Identification des coexistences
  coex <- ifelse(spA > 0 & spB > 0, 1, 0)
  
  #Idem pour A qui arrive en premier :
  spA_A1 <- 100-DA-(0.5*CB)
  spB_A1 <- 100-DB-(1.5*CA)
  
  coex_A1 <- ifelse(spA_A1 > 0 & spB_A1 >0, 1, 0)
  
  #Idem pour B qui arrive en premier :
  spA_B1 <- 100-DA-(1.5*CB)
  spB_B1 <- 100-DB-(0.5*CA)
  
  coex_B1 <- ifelse(spA_B1 > 0 & spB_B1 > 0, 1, 0)
  
  #Identification des situations o il y a coexistence :
  EP2_N <- paste0(coex,coex_B1,coex_A1)
  library(Renaudpack2)
  EP22 <- Remplacer(EP2_N,VALEURS = c("000", "001", "101", "111", "110", "010"),
                   VALEURS_NEW = c("Jamais","Si A1","Sauf B1","Toujours","Sauf A1","Si B1"))
  EP22[EP22=="Jamais"] <- ifelse(spA[EP22=="Jamais"] > 0, "A seule","B seule")
  EP22_col <- Remplacer(EP22, VALEURS = c("B seule", "Si A1","Sauf B1","Toujours", "Sauf A1","Si B1", "A seule"), VALEURS_NEW = coul)
  plot(Equi, Assy, pch = 16, col = EP22_col, xlim = xlim, main = main, ...)
  legend("topright", 
         legend = c("B seule", "Coex. si A 1ere","Coex sauf B 1ere","Toujours coex.", "Coex. sauf A 1ere","Coex. sauf si B 1ere", "A seule"),
         bty = "n", col = coul, pch=16)
}
Coexistence_plot2(DA = DA ,DB = DB, CA = CA, CB = CB, xlim=c(-1,1.5))
